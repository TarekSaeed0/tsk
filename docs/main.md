## Features

- Generic data structures implemented in C, allowing you to work with any data type.
- Consistent and easy-to-use API for common data structures.
- Working with generic types through traits.

## Building

```sh
git clone https://github.com/TarekSaeed0/tsk.git
cd tsk
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```
